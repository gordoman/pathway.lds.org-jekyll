Calculate Tuition

Tuition is based on location and the number of BYU-Idaho course credits a student takes each semester.

COURSE CREDIT:

A measurement used to quantify the load of work for a specific course that enables universities to set a requirement to graduate.

Pathway standard version courses are either 2 or 3 credits each.
